# Blog Post Ideas

A running list of blog post topics. Prioritized by interest and relevance. See [[MOC - Knowledge Management]] for the broader content strategy.

## High Priority

### 1. "Migrating from jQuery to React: A Pragmatic Guide"
- Draw from real experience with [[Projects/Web App Redesign]]
- Cover the micro-frontend approach for incremental migration
- Include code examples showing before/after
- **Target:** 2500 words | **Audience:** Frontend developers maintaining legacy codebases
- **Status:** Outline drafted

### 2. "Building a Terminal UI with Go and Bubble Tea"
- Behind-the-scenes of building Granit
- Cover the Elm architecture (Model-Update-View) in Go
- Show how overlays, focus management, and async commands work
- **Target:** 3000 words | **Audience:** Go developers interested in TUI
- **Status:** Notes collected

### 3. "Local-First AI: Running LLMs in Your Dev Tools"
- How Granit integrates Ollama for local AI features
- Compare local vs cloud AI for developer tools
- Privacy benefits, latency tradeoffs, model selection
- **Target:** 2000 words | **Audience:** Developers interested in AI tooling
- **Status:** Idea stage

## Medium Priority

### 4. "Graph Databases for Personal Knowledge Management"

- Data portability and longevity arguments
- Compare Notion, Obsidian, Granit approaches to data ownership

- **Target:** 1500 words

### 6. "Effective Code Review: Beyond Nitpicking"
- Patterns from our team's review process
- How to give actionable feedback
- Setting up review checklists
- **Target:** 1800 words
