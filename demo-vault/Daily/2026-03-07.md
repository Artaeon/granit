### 11:00 — Bug Fix

Fixed a tricky bug in the sidebar where the file tree wasn't properly collapsing nested folders when the parent was toggled. The issue was that the `toggleFolder` function wasn't recursively updating child visibility states.

Root cause: the tree traversal was using a breadth-first approach but the visibility update needed depth-first to properly cascade the collapsed state downward.
