## About This Vault

This demo vault contains **18 interconnected notes** spanning projects, research, daily logs, meetings, ideas, and book notes. It demonstrates:

- **Wikilinks** for connecting ideas across notes
- **YAML frontmatter** for metadata and tags
- **Task management** with priorities and due dates
- **Code blocks** with syntax highlighting
- **Tables** for structured comparisons
- **Mermaid diagrams** for visual documentation
- **Templates** for repeatable workflows
