# Keyboard Shortcuts

## Navigation
- `Ctrl+1` — Focus sidebar
- `Ctrl+2` — Focus editor
- `Ctrl+3` — Focus backlinks

## Editing
- `Ctrl+S` — Save
- Arrow keys or `j`/`k` — Move cursor
- `Enter` — Open file or new line

## Search
- Type in sidebar to fuzzy search
- `Backspace` to clear search

## Panels
- `Tab` — Toggle between backlinks and outgoing links
