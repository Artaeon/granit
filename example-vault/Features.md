# Features

## Vault Compatibility
Granit reads any [[Welcome|Obsidian vault]] without modification.

## WikiLinks
Full support for [[Welcome|wikilinks]] including aliases.

## Backlinks
See which notes link to the current one — the "graph in text."

## Fuzzy Search
Find any file instantly with fzf-like fuzzy matching.

## Markdown Editor
Edit notes directly in the terminal with syntax highlighting.

## Daily Notes
Create and open daily notes with a single command.
See [[Daily Notes]] for more details.

## Cross-Platform
Runs on Linux, macOS, and Windows.
