# Writing Workflow

My process for turning notes into published writing.

## Pipeline

### 1. Capture
- Jot ideas in [[Daily Notes]]
- Use fleeting notes for raw thoughts
- Clip articles with the web clipper

### 2. Process
- Review captured notes weekly
- Create permanent notes using [[Zettelkasten Method]]
- Link to existing knowledge

### 3. Draft
- Use **Thread Weaver** to synthesize notes
- Write in **Zen mode** for focus
- Let **Ghost Writer** suggest continuations

### 4. Edit
- Toggle to **view mode** to read as a reader
- Check structure with **note outline**
- Review backlinks for missing connections

### 5. Publish
- Export to HTML or PDF
- Use the **static site publisher** for web

## Tools in Granit

- [x] Distraction-free Zen mode
- [x] AI Ghost Writer for suggestions
- [x] Thread Weaver for synthesis
- [x] Export to HTML/PDF
- [ ] Word count goals (planned)

## Tips

> Write **fast**, edit **slow**. Capture everything, curate later.
