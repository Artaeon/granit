# Building a Second Brain

A system for organizing digital information using the **PARA method**:

## PARA Framework

- **Projects** — Short-term efforts with a goal and deadline
- **Areas** — Long-term responsibilities to maintain
- **Resources** — Topics of ongoing interest
- **Archives** — Inactive items from the above

## Key Principles

1. **Capture** everything that resonates
2. **Organize** for actionability, not by topic
3. **Distill** progressive summarization
4. **Express** turn knowledge into output

## Implementation in Granit

Granit makes this workflow natural:
- [[Welcome|Wikilinks]] connect ideas across notes
- [[Features|Fuzzy search]] finds anything instantly
- Tags organize notes by project and area
- [[Daily Notes]] capture fleeting thoughts
