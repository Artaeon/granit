# Zettelkasten Method

The **Zettelkasten** (German for "slip box") is a note-taking method developed by sociologist Niklas Luhmann.

## Core Rules

1. **Atomic notes** — Each note contains one idea
2. **Link generously** — Every note connects to others
3. **Write in your own words** — No copy-paste
4. **Use unique identifiers** — Permanent addresses for notes

## Types of Notes

| Type | Purpose | Example |
|------|---------|---------|
| **Fleeting** | Quick capture | Daily journal entries |
| **Literature** | Book/article notes | Reading summaries |
| **Permanent** | Your own ideas | Original insights |
| **Structure** | Maps of content | Index notes |

## Why It Works

- Forces **active thinking** over passive collection
- Creates an **interconnected web** of knowledge
- Enables **serendipitous discovery** of connections
- Scales infinitely without reorganization
